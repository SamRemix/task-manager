# task-manager-MERN

**Create .env file in back_ folder**

PORT
> same as package.json proxy in front_ folder (4000)

MONGO_URI
> mongodb+srv://[username:password@]host[/[database][?options]]

SECRET
> jwt secret key

```
npm i -g nodemon

cd back_
npm i
npm run dev

cd front_
npm i
npm run start
```
